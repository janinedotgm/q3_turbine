## Setup

Before you start, ensure you have a `wba-wallet.json` file in the `wallet` directory.

## Running the Scripts

1. Install the project dependencies:
    ```sh
    yarn
    ```

2. Execute the scripts using:
    ```sh
    yarn <script-name>
    ```
